# Launch Java easily
This bash script permits to compile Java files easily and automatically launch

## Usage
```
https://javacompile.thomaslacaze.fr/javaCompile.sh && chmod +x ./javaCompile.sh && ./javaCompile.sh fileToCompile.java 
```
